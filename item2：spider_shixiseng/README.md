	    员工人数：company_workers

	    公司联系人：company_ecname

	    公司传真：company_fax
	    公司地址：company_address

	    电子邮箱：company_email
	    公司描述：company_detail

	    job_coreskills  varchar(256) NOT NULL技能关键字
